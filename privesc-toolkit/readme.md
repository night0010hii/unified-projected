# Linux PrivEsc Toolkit

> FOR EDUCATIONAL AND AUTHORIZED USE ONLY

## Quick Start

### Backend

cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

### Dashboard (no build needed)

Open frontend/dashboard.html in your browser
Click Start Scan

### CLI

cd scripts
python3 cli_scan.py
python3 cli_scan.py --output report.html

### React (optional)

cd frontend
npm install
npm start
