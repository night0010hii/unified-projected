# 🚀 Unified Mentor Internship Projects

This repository contains a collection of projects developed during my internship at Unified Mentor.  
These projects demonstrate practical implementation of cybersecurity concepts, system monitoring, and full-stack development.

---

## 📂 Projects Included

### 🔹 1. Pass Audit Tool
- Analyzes password strength and security
- Includes entropy calculation and brute-force simulation
- Tech: Python

### 🔹 2. Privilege Escalation Toolkit
- Scans Linux systems for privilege escalation vulnerabilities
- Includes multiple security checks (SUID, sudo, cron, etc.)
- Tech: Python, Flask, React

### 🔹 3. Registry Monitor
- Monitors system registry changes
- Detects suspicious modifications
- Tech: Python

### 🔹 4. Secure File Monitor
- Tracks file integrity and detects unauthorized changes
- Generates logs and alerts
- Tech: Python

### 🔹 5. Sentinel Shield (Web Protection System)
- Basic intrusion detection and web protection system
- Includes logging and WAF-like functionality
- Tech: JavaScript, HTML, CSS

### 🔹 6. Threat Intelligence Aggregator
- Collects and processes threat intelligence feeds
- Generates blocklists and reports
- Tech: Python, Flask
