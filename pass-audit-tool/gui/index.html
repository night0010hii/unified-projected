MODULE DESCRIPTIONS
dictionary_generator.py
Builds targeted wordlists from keywords with leet, upper, suffix, prefix mutations. Supports DOB + name combos.
hash_extractor.py
Parses Linux /etc/shadow and Windows SAM (offline). Identifies MD5, SHA-512, bcrypt, NTLM algorithms.
bruteforce_simulator.py
Simulates dictionary and incremental attacks. Estimates real GPU crack times. Ethical limits enforced.
strength_analyzer.py
Scores 0–100. Checks length, charset diversity, dictionary membership, entropy bits.
entropy_calculator.py
Shannon entropy per character and total bits. Charset size analysis. Frequency mapping.
hash_identifier.py
Auto-detects hash type from length and format patterns. Covers 15+ algorithms.
policy_checker.py
Validates against configurable rules: min length, charset requirements, common password blocklist.
report_generator.py
Produces full audit report in .txt and .json with severity ratings and policy recommendations.
